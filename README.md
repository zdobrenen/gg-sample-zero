# AWS Greengrass Sample Zero

A sample project intended to demonstrate an in-depth yet simple utilization of AWS Cloud Services in
IoT.

## AWS Services

- AWS Greengrass
- AWS IoT
- AWS Lambda
- AWS Kinesis Firehose
- AWS S3
- AWS RDS
- AWS API Gateway
- AWS Amplify
- AWS Cognito
